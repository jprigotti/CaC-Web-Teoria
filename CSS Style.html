Declaración de estilos en CSS

En la etiqueta HTML

<body>
    <main>
        <h2 style="color:green">Hay 3 formas de usar CSS</h2>
    </body>
</main>

En el cuerpo del archivo HTML

<head>
    <style>
        h2{
        color: blue;
        }
    </style>
</head>

Llamando un archivo (local/externo)

<head>
    <!-- Inserting Icons from Fontawesome -->
    <script src="https://kit.fontawesome.com/badcd6f1fd.js" crossorigin="anonymous"></script>
                
    <!-- Local CSS Style -->
    <link rel="stylesheet" href="./CSS/style.css">
</head>

El estilo se puede aplicar a:

  • Un elemento (en el CSS usamos simplemente el elemento)
  • Una Class (en el CSS usamos .Class)/li>
  • Un ID (en el CSS usamos #ID)
// Aplicamos estilo al elemento//
p{
    color:#9e44ae;
}

// Aplicamos estilo a la clase//
.class_1{
    color:#a877b1;
}

// Aplicamos estilo al ID//
#id_1{
    color: aqua;
}

Peudo classes & pseudo elements

En los siguientes ejemplos de código, podemos ver el uso de uno y otro.

Pseudo classes definen estilo de un estado, por ejemplo el hover de un link

// Aplicamos estilo al estado hover de un link
.cssNav_2__item:hover{
background: #1D2F3D;
color: #fff;
text-decoration: none;
    }

mientras que los Pseudo elements definen una parte específica, ::before ::after 

/*Pseudo element*/
.cssNav_1_Ul__item a::before{
    content: " complement text ";
}
